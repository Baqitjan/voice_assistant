# LOQ-GENESIS 🤖 — Жеке ИИ-Агент

> Lenovo LOQ (RTX 4060) үшін арнайы бапталған, Windows 11 жүйесін дауыспен басқаруға арналған модульдік агент.

---

## 🚀 Жүйелік талаптар

| Компонент | Талап |
|-----------|-------|
| **ОЖ** | Windows 11 |
| **GPU** | NVIDIA RTX 4060 (8GB VRAM) немесе жоғары |
| **Python** | 3.10 немесе 3.11 |
| **CUDA** | 12.4 (Faster-Whisper және PyTorch үшін) |

---

## 📂 Жоба құрылымы

```
loq-genesis/
├── main.py                  # Басты хаб (Оқиғалар циклі)
├── requirements.txt         # Барлық қажетті кітапханалар
├── install.bat              # Автоматты орнату файлы
├── config/
│   └── settings.py          # Баптаулар (Константалар, Промпттар)
├── engine/
│   ├── ear.py               # Ear: VAD + Faster-Whisper (GPU)
│   ├── brain.py             # Brain: Ollama (Gemma 2 / Llama 3)

│   └── vision.py            # Vision: Скриншот + LLaVA
├── tools/
│   ├── pc_control.py        # Windows басқару (PyAutoGUI)
│   ├── web_search.py        # DuckDuckGo Интернет іздеу
│   └── scheduler.py         # Еске салғыштар (Scheduler)
├── data/
│   ├── my_voice.wav         # Дауысты клонтауға арналған референс
│   └── screenshot.png       # Уақытша скриншоттар
└── logs/                    # Жүйелік логтар
```

## 🛠 Орнату және Баптау

### 1. Виртуалды ортаны дайындау

```bash
python -m venv venv
.\venv\Scripts\activate
.\install.bat
```

### 2. Ollama модельдерін жүктеу

Ассистент дұрыс жұмыс істеуі үшін мына модельдерді алдын ала орнатыңыз:

```bash
ollama pull gemma2:9b        # Негізгі ойлау моделі (Қазақша жақсы түсінеді)
ollama pull llava:7b         # Экранды көру (Vision) моделі
```

### 3. Kokoro TTS файлдары

`kokoro-v0_19.onnx` және `voices.bin` файлдарын жобаның түбіріне (root) қойыңыз.  
Бұл дауыстың өте табиғи шығуына жауап береді.

## 🗣 Дауыстық командалар

| Команда | Сипаттама |
|---------|-----------|
| `"Ассистент"` | Агентті ояту (Wake-word) |
| `"Экранда не бар?"` | Экранды скриншоттап, LLaVA арқылы талдау |
| `"Блокнотты аш"` | Windows қолданбасын іске қосу |
| `"Блокнотқа [мәтін] деп жаз"` | Мәтінді автоматты теру |
| `"Гуглдан [сұрақ] ізде"` | Интернеттен ақпарат тауып беру |
| `"Еске сал [уақыт]"` | Таймер немесе еске салғыш қою |
| `"Тазала"` / `"Ұмыт"` | Диалог тарихын өшіру |

---

## 🤖 Action-Командалар форматы

LLM жүйені басқару үшін мынадай JSON қайтарады:

```
[ACTION: {"cmd": "open_app", "args": "notepad"}]
```

**Қолдау көрсетілетін командалар:**

| Команда | Сипаттама |
|---------|-----------|
| `open_app`, `close_app` | Бағдарламаларды басқару |
| `type_text`, `press_key`, `hotkey` | Пернетақтамен жұмыс |
| `screenshot` | Vision модулін шақыру |
| `run_cmd` | Терминал арқылы команда орындау |

---

## ⚠️ Маңызды ескертулер

- **Микрофон:** Сапалы транскрипция үшін шусыз орта немесе жақсы микрофон қажет.
- **GPU:** Егер GPU жады жетпей жатса (8GB аздау болуы мүмкін), Ollama-да модельдерді кезекпен жүктеуді баптаңыз.
- **Қауіпсіздік:** `run_cmd` командасын қолданғанда абай болыңыз — ИИ маңызды файлдарды жойып жібермеуі керек.
