# LOQ-GENESIS 🤖 — Персоналды ИИ-Агент

> Windows 11 жүйесін дауыспен басқаруға арналған, NVIDIA RTX 4060 қуатын қолданатын модульдік агент.  
> Бұл агент **қазақ және орыс тілдеріндегі** бұйрықтарды түсініп, компьютерде әрекеттер орындайды.

---

## 🚀 Жүйелік сипаттама

| Компонент | Мән |
|-----------|-----|
| **ОЖ** | Windows 11 |
| **GPU** | NVIDIA RTX 4060 (8GB VRAM) — Faster-Whisper және Ollama жеделдету үшін міндетті |
| **Python** | 3.10 / 3.11 |
| **Негізгі модель** | Llama 3.1:8b (Ollama арқылы) |

---

## 📂 Жоба құрылымы

```
loq-genesis/
├── main.py                  # Негізгі цикл (Аудионы қабылдау -> Ойлау -> Орындау)
├── install.bat              # Бір басумен барлық драйверлерді орнату
├── config/
│   └── settings.py          # Промпттар, модель аттары және сезімталдық баптаулары
├── engine/
│   ├── ear.py               # Ear: Faster-Whisper (GPU) — дауысты мәтінге айналдыру
│   ├── brain.py             # Brain: Ollama (Llama 3.1) — мағынаны түсіну және JSON генерациясы
│   ├── mouth.py             # Mouth: Kokoro / Silero TTS — жауапты дыбыстау
│   └── vision.py            # Vision: LLaVA — экрандағы бейнені талдау
└── tools/
    ├── pc_control.py        # Windows басқару (Apps, Keys, Mouse)
    └── web_search.py        # Интернеттен ақпарат іздеу
```

## 🛠 Орнату және Баптау

### 1. Ортаны дайындау

Жоба папкасында терминалды ашып, мынаны іске қос:

```powershell
.\install.bat
```

### 2. Модельдерді жүктеу (Ollama)

Сен қолданып жатқан модельдерді Ollama-ға тартып ал:

```powershell
ollama pull llama3.1:8b     # Негізгі ақыл-ой
ollama pull llava:7b        # Көру модулі
```

### 3. Дауыс файлдары

`kokoro-v0_19.onnx` және `voices.bin` файлдары жобаның басты папкасында тұруы тиіс.

## 🗣 Негізгі командалар

| Команда | Нәтиже |
|---------|--------|
| `"Ассистент"` | Агентті белсендіру (Wake-word) |
| `"Блокнотты аш"` | `open_app("notepad")` командасын орындайды |
| `"Блокнотқа [текст] деп жаз"` | Қазақша/орысша мәтінді тереді |
| `"Экранда не бар?"` | Экранды көріп, мазмұнын айтып береді |
| `"Интернеттен [іздеу] тап"` | DuckDuckGo арқылы іздеу салады |

---

## 🤖 Action-Logic (Компьютерді басқару)

`Llama 3.1:8b` моделіне келесі форматта жауап беру бұйырылған:

```
[ACTION: {"cmd": "open_app", "args": "notepad"}]
```

**Қолжетімді командалар:**

| Команда | Сипаттама |
|---------|-----------|
| `type_text` | Мәтін енгізу |
| `press_key` / `hotkey` | Пернелер комбинациясы |
| `screenshot` | Экранды суретке түсіру |
| `run_cmd` | Терминал командалары |

---

## ⚠️ Маңызды баптаулар (`settings.py`)

- **Whisper:** Егер қазақша түсінбесе, `WHISPER_LANGUAGE = None` (автоматты анықтау) мәніне қойыңыз.
- **Brain:** `LLM_DEFAULT = "llama3.1:8b"` екеніне көз жеткізіңіз.
- **VAD:** Егер ассистент сөзіңді бөліп жіберсе, `SILENCE_DURATION_SEC = 3.0` мәнін көтеріңіз.
